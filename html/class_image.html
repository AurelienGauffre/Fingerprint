Detailed Description

class of the image we'll work on

A Pixel represent a 2D Points which float coordinates which has an intenisty. Array of Pixels are used the class Image when when dealing with non integer coordinates, mostly during interpolations. The basic geometrical transformation are implemented for Pixel such as translation, rotation and warping.

The class is composed of several methods
